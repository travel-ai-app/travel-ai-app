# AI Travel Companion – Product Spec (v1)

## 1. Τι είναι

Ένα mobile app που λειτουργεί σαν **AI ταξιδιωτικός συνοδοιπόρος**.  
Δεν είναι μόνο πρόγραμμα ταξιδιού, ούτε μόνο έξοδα, αλλά ένας βοηθός που σου λέει:

> «Τι να κάνω τώρα; Πού να πάω; Πόσα ξόδεψα;»

---

## 2. Βασικά Features (MVP)

1. **Δημιουργία Ταξιδιού (Trip Setup)**
   - Προορισμός, ημερομηνίες, τύπος ταξιδιού (solo, couple, friends, family)
   - Ενδιαφέροντα (φαγητό, παραλίες, nightlife, sightseeing κλπ.)
   - Προαιρετική καταχώρηση πτήσεων, ξενοδοχείου, κλεισμένων tours

2. **AI “Τι να κάνω τώρα;” (Real-time suggestions)**
   - Παίρνει τοποθεσία, ώρα, καιρό, προτιμήσεις
   - Φέρνει κοντινές επιλογές (φαγητό, δραστηριότητες, βόλτες)
   - Χρησιμοποιεί reviews από επίσημα APIs (rating, αριθμός reviews, σχόλια)
   - Συγκρίνει και προτείνει τις καλύτερες με AI
   - Δίνει σύντομη εξήγηση “γιατί το προτείνει”

3. **Δυναμικό Πρόγραμμα Ταξιδιού (Itinerary)**
   - Προβολή μέρας-μέρας (π.χ. Day 1, Day 2…)
   - Σταθερά events (πτήσεις, hotel check-in, tours)
   - Κενά slots που μπορεί να γεμίσει το AI με προτάσεις
   - Μελλοντικά: αυτόματη προσαρμογή σε αλλαγές (καιρός κλπ.)

4. **Καταγραφή Εξόδων Ταξιδιού (Expenses)**
   - Ο χρήστης καταχωρεί έξοδα (ποσό, κατηγορία, ημερομηνία)
   - Δεν υπάρχει υποχρεωτικό budget
   - Στο τέλος του ταξιδιού: σύνολο, ανά μέρα, ανά κατηγορία, μέσος όρος

5. **AI OCR απόδειξης (Auto expense via photo)**
   - Ο χρήστης βγάζει φωτογραφία απόδειξης
   - Το AI διαβάζει ποσό + currency
   - Προτείνει κατηγορία (φαγητό, μετακίνηση κτλ.)
   - Συνδέει το έξοδο με την ημέρα/δραστηριότητα

---

## 3. Τεχνικά (υψηλό επίπεδο)

- Mobile app (προς το παρόν Flutter, θα επιβεβαιωθεί)
- Backend: Firebase / Firestore
- AI: OpenAI API (προτάσεις + OCR + ταξινόμηση reviews)
- Εξωτερικά APIs: π.χ. Google Places ή αντίστοιχο για μέρη + reviews

---

## 4. Γλώσσες

- Πρώτη έκδοση: Αγγλικό UI
- Όλα τα κείμενα σε localization files
- Μελλοντικά: bulk AI μετάφραση σε Ελληνικά (όχι manual string-by-string)
